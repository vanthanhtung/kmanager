## Overview

**Page Purpose:** Give TГ№ng a complete overview of every venue on the platform вҖ” which are active, when they were last used, and quick metrics вҖ” and let him create, edit, or disable venues from one screen.

**User Situation:** A new venue owner in District 2 wants to use kmanager. TГ№ng needs to create their venue, set up their manager account, and hand over login credentials. He also notices one venue hasn't been active in 3 weeks вҖ” time to check in.

**Success Criteria:** Create a new venue in under 2 minutes. See all venues and their status at a glance. Identify inactive venues immediately.

**Entry Points:**
- Super admin login вҶ’ lands here
- Super admin URL (separate from venue login)

**Exit Points:**
- Click venue row вҶ’ Venue Management (03.2) edit mode
- Click "Add Venue" вҶ’ Venue Management (03.2) create mode
- Logout вҶ’ super admin login page

## Technical Notes

- Super admin is a separate login flow from venue manager
- Super admin can see ALL venues across the platform
- Venue data is lightweight (counts, not full details) for fast loading
- Last activity = timestamp of most recent bill close or check-in
- Super admin cannot access venue billing data вҖ” only venue management metadata
